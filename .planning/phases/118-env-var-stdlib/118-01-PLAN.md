```rust
#[no_mangle]
pub extern "C" fn mesh_env_get(key: *const MeshString) -> *mut MeshOption {
    unsafe {
        let key_str = (*key).as_str();
        match std::env::var(key_str) {
            Ok(val) => {
                let s = mesh_string_new(val.as_ptr(), val.len() as u64);
                alloc_option(0, s as *mut u8)
            }
            Err(_) => alloc_option(1, std::ptr::null_mut()),
        }
    }
}
```

In `crates/mesh-rt/src/env.rs`, add two new `#[no_mangle] pub extern "C"` functions after the existing `mesh_env_get`:

**mesh_env_get_with_default(key: *const MeshString, default: *const MeshString) -> *mut MeshString:**
- Read `(*key).as_str()` and call `std::env::var(key_str)`
- On `Ok(val)`: call `mesh_string_new(val.as_ptr(), val.len() as u64)` and return it
- On `Err(_)`: return `default as *mut MeshString` (the caller-provided default is already a MeshString pointer)
- Both branches are `unsafe` blocks — the whole body should be `unsafe { ... }`

**mesh_env_get_int(key: *const MeshString, default: i64) -> i64:**
- Read `(*key).as_str()` via `unsafe { (*key).as_str() }`
- Call `std::env::var(key_str).ok()` to get `Option<String>`
- On `Some(val)`: call `val.parse::<i64>().unwrap_or(default)`
- On `None`: return `default`
- The parse failure path (non-numeric, overflow) silently returns default — no stderr output required
