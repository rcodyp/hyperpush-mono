Add ~r/pattern/flags regex literal syntax to the Mesh lexer, parser, AST, type checker, and MIR lowerer.

Purpose: Regex literals allow users to write compile-time constant regex patterns. The literal desugars to a runtime call `mesh_regex_from_literal(pattern_str, flags_bitmask)` that compiles the pattern once and returns an opaque `Regex` pointer.
Output: New RegexLiteral token, REGEX_EXPR AST node, and full pipeline from source text to MIR call.
</objective>

Add a private method `fn lex_regex_literal(&mut self, start: u32) -> Token`:
- Consume `r` (already peeked, now advance cursor).
- Expect `/`; if absent, return `TokenKind::Error`.
- Read characters until an unescaped `/` is found (handle `\/` as literal slash in pattern — keep the backslash+slash in the pattern string). If EOF before closing `/`, return `TokenKind::Error`.
- After closing `/`, read zero or more flag chars: only `i`, `m`, `s` are valid flags. Stop at first non-flag, non-identifier character. If any other letter is found (e.g., `g`, `x`), return `TokenKind::Error` with a descriptive message logged (but still emit Error token, don't panic).
- Return `Token::new(TokenKind::RegexLiteral(pattern, flags), start, self.cursor.pos())`.

Pattern content: collect chars from after the opening `/` up to (but not including) the closing unescaped `/`. Do NOT unescape `\/` — keep it as-is so the runtime regex library sees the correct pattern. Example: `~r/hello\/world/` -> pattern=`"hello\/world"`, flags=`""`.

Define the `RegexExpr` struct and impl below the other ast_node! impls:
```rust
ast_node!(RegexExpr, REGEX_EXPR);
impl RegexExpr {
    /// Returns the regex pattern string (content between `/` delimiters).
    pub fn pattern(&self) -> Option<String> {
        self.syntax.children_with_tokens()
            .filter_map(|c| c.into_token())
            .find(|t| t.kind() == SyntaxKind::REGEX_LITERAL)
            .and_then(|t| {
                // Token text is the source text like ~r/pat/flags
                // We need to extract just the pattern from the TokenKind payload.
                // Use the raw token text and parse it out.
                let text = t.text();
                // text looks like: ~r/pattern/flags
                if let Some(rest) = text.strip_prefix("~r/") {
                    if let Some(slash_pos) = rest.find('/') {
                        return Some(rest[..slash_pos].to_string());
                    }
                }
                None
            })
    }

    /// Returns the flags string (e.g. "ims", "i", "").
    pub fn flags(&self) -> String {
        self.syntax.children_with_tokens()
            .filter_map(|c| c.into_token())
            .find(|t| t.kind() == SyntaxKind::REGEX_LITERAL)
            .map(|t| {
                let text = t.text();
                if let Some(rest) = text.strip_prefix("~r/") {
                    if let Some(slash_pos) = rest.find('/') {
                        return rest[slash_pos + 1..].to_string();
                    }
                }
                String::new()
            })
            .unwrap_or_default()
    }
}
```

NOTE: The `TokenKind::RegexLiteral(pattern, flags)` data fields aren't directly accessible from SyntaxToken (the CST stores just the text). The `pattern()` and `flags()` methods parse the source text `~r/pat/flags`. This is fine — it is the same pattern used for `AtomLiteral::atom_text()` which reads the `:atom` source text. Make sure to handle escaped slashes correctly in `pattern()`: the find('/') approach may break if the pattern contains `\/`. Use the simpler approach: since we store the whole text as `~r/{pattern}/{flags}`, split at the LAST `/` (since pattern can't contain a literal unescaped `/`), or use the fact that `\/` is two chars and find the first `/` that is NOT preceded by `\`.

Simpler: split on the closing `/` by scanning: after `~r/`, walk char by char, tracking `prev_was_backslash`. First unescaped `/` is the closing delimiter.

```rust
Expr::RegexExpr(rx) => {
    // Regex literal desugars to a mesh_regex_from_literal(pattern, flags_bitmask) call.
    // mesh_regex_from_literal panics on invalid pattern (compile-time constant, must be valid).
    let pattern = rx.pattern().unwrap_or_default();
    let flags_str = rx.flags();
    // Compute flags bitmask: i=1, m=2, s=4
    let flags_bits: i64 = flags_str.chars().fold(0i64, |acc, c| match c {
        'i' => acc | 1,
        'm' => acc | 2,
        's' => acc | 4,
        _ => acc,
    });
    let fn_ty = MirType::FnPtr(
        vec![MirType::String, MirType::Int],
        Box::new(MirType::Ptr),
    );
    MirExpr::Call {
        func: Box::new(MirExpr::Var("mesh_regex_from_literal".to_string(), fn_ty)),
        args: vec![
            MirExpr::StringLit(pattern, MirType::String),
            MirExpr::IntLit(flags_bits, MirType::Int),
        ],
        ty: MirType::Ptr,
    }
}
```
