Implement:

**`mesh_regex_from_literal(pattern: *const MeshString, flags_bits: i64) -> *mut u8`**
- Build regex with flags from bitmask. Panic with descriptive message on compile error.
- `flags_bits & 1 != 0` = case_insensitive, `& 2 != 0` = multi_line, `& 4 != 0` = dot_matches_new_line
- Return `Box::into_raw(Box::new(compiled)) as *mut u8`

**`mesh_regex_compile(pattern: *const MeshString) -> *mut MeshOption`**
- Try to compile pattern. Returns MeshResult (using MeshOption layout: tag=0 Ok, tag=1 Err).
- On success: create Box<regex::Regex>, return alloc_option(0, Box::into_raw(Box::new(rx)) as *mut u8)
- On failure: create MeshString from error message, return alloc_option(1, error_str as *mut u8)
- Use `alloc_option` from crate::option for the Result wrapper (same ABI as Option — both are tag+value unions).

**`mesh_regex_match(rx_ptr: *const u8, s: *const MeshString) -> bool`**
- Cast rx_ptr to `*const regex::Regex` (via `&*(rx_ptr as *const regex::Regex)`).
- Return `rx.is_match(s_str)`.

**`mesh_regex_captures(rx_ptr: *const u8, s: *const MeshString) -> *mut MeshOption`**
- Use `rx.captures(s_str)`.
- If None: return `alloc_option(1, std::ptr::null_mut())` (None).
- If Some(caps): build a MeshList of the capture group strings (index 0 is the whole match, indices 1..n are groups). Use `mesh_list_new()` and `mesh_list_append()` (imported from crate::collections). Return `alloc_option(0, list_ptr)`.

Look at how mesh_string_split in string.rs builds a list — use the same pattern. If `mesh_list_new` and `mesh_list_append` are available as extern fns or Rust-callable fns, use them. If they are only extern "C", call them via unsafe ffi from within the same crate (they are in-crate, so just call as Rust functions).

**`mesh_regex_replace(rx_ptr: *const u8, s: *const MeshString, replacement: *const MeshString) -> *mut MeshString`**
- Use `rx.replace_all(s_str, repl_str).into_owned()`.
- Return `mesh_string_new(result.as_ptr(), result.len() as u64)`.

**`mesh_regex_split(rx_ptr: *const u8, s: *const MeshString) -> *mut u8`**
- Use `rx.split(s_str).collect::<Vec<&str>>()`.
- Build a MeshList of MeshString pointers for each part.
- Return the list pointer as `*mut u8`.

Add unit tests in a `#[cfg(test)] mod tests` block:
